# pdfbot

для запуска вставьте токен от ботфазера в .env, соберите образ и можете запускать

из коробки бот не шпионит, но прикрутить шпионилку в перспективе не сложно

Опционально можно настроить вебхук:
WEBHOOK_URL=https://example.com/telegram/webhook
WEBHOOK_HOST=0.0.0.0
WEBHOOK_PORT=8080

Если WEBHOOK_URL не задан, бот удаляет вебхук у Telegram и работает через polling.
